# ✅ 到公司後待辦清單

## □ 步驟 1：找到 GDrive 路徑

```powershell
Test-Path "G:\我的雲端硬碟"
Test-Path "$env:USERPROFILE\Google Drive"
Test-Path "$env:USERPROFILE\My Drive"
```

找到後記下來：`<公司GDrive> = _____________`

---

## □ 步驟 2：建立專案資料夾並拉 GitHub

```powershell
mkdir -p "<公司GDrive>\水電管理"
cd "<公司GDrive>\水電管理"
git init
git remote add origin https://github.com/jems06212-dotcom/water-electricity-mgmt
git fetch origin
git checkout -b master origin/master
```

---

## □ 步驟 3：把 Codex 寫好的檔案搬進來

```powershell
# 把 Documents\Codex 裡跟水電管理有關的檔案複製過來
Copy-Item -Path "$env:USERPROFILE\Documents\Codex\*水電*" -Destination "." -Recurse

# 如果有其他相關檔案，手動搬過來
# 或者直接把整個目錄結構建好後手動貼上
```

---

## □ 步驟 4：git commit + push

```powershell
git add .
git commit -m "feat: 匯入 Codex 撰寫的專案文件"
git push origin master
```

---

## □ 步驟 5：更新 .codex/AGENTS.md 路徑

編輯專案內的 `.codex\AGENTS.md`，把 `D:\googole driver` 改成你的 GDrive 路徑。

---
