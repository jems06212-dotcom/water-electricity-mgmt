# ✅ 到公司後 — 兩步驟

## 第 1 步：建立環境一致性

打開 `ENVIRONMENT.md`，照**第十章檢查清單**全部跑完。

要裝的東西一次全列在裡面了：
- Node.js、Git、GitHub CLI、Python
- 16 個 npm 全域套件（`npm install -g ...`）
- Python pip（notebooklm-mcp-cli）
- 編輯 `config.toml`（範本在 `_opencode-bundle\config.toml.template`）
- 複製 CLAUDE.md + .omo → D:\OPENCODE_0621
- 設定 6 個環境變數
- 3 次登入（gh、codex、firebase）

**全部大約 15-20 分鐘。**

---

## 第 2 步：接上專案 + 搬 Codex 檔案

環境都一樣了之後：

```powershell
# 1. git clone 或進入水電管理專案
cd "<公司GDrive路徑>\水電管理"
git pull origin master

# 2. 把 Codex 在 Documents\Codex 寫好的檔案搬過來
Copy-Item -Path "$env:USERPROFILE\Documents\Codex\*水電*" -Destination "." -Recurse

# 3. 如果有新的檔案就 commit + push
git add .
git commit -m "feat: 匯入 Codex 撰寫的專案文件"
git push origin master
```
